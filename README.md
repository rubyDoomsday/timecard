# TimeCard

Small project to automate time tracking in SpringAhead

## Prerequisites

| dependency | version |
| ---        | ---     |
| Python3    | >=3.11  |
| pip        | >=23.0  |
| homebrew   | >=1.3.0 |

## Installation

This project uses STRTA to mange dependencies

```sh
script/bootstrap
```

## Usage

```sh
script/run
```
